- Functional anarchy

> I would sort all these numbers for you, but you have no authority over me.
> - Unknown anarchist

In this challenge you have to read multiple integers from the standard input
and sort them in ascending order.

- Expected input:
In the first line, a single integer of size N, representing the amount 
of lines to read, followed by N lines contain an integer each.

- Expected output:
Print all the numbers in ascending order.

- Sample input:
5
7
3
5
1
6

- Sample output:
1
3
5
6
7

- Test cases:
All the test cases are in the 'tests' folder.
