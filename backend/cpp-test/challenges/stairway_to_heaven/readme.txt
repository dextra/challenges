- Stairway to heaven

> There's a coder who's sure all that glitters is gold
> And she's printing the stairway to heaven
> - Led Zeppelin

In this challenge you have to read an integer from the standard input and
print a staircase.

- Expected input:
A single integer of size N, representing the size of the staircase.

- Expected output:
Print a staircase of size N using # and spaces.

- Constraints:
Input: 0 < N
Output: A staircase or "Error." if the input is invalid

- Sample input:
3

- Sample output:
  #
 ##
###

- Test cases:
All the test cases are in the 'tests' folder.
