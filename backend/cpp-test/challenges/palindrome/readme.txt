- Palindrome

A palindrome is a word, phrase, number, or other sequence of characters 
which reads the same backward or forward, such as 'madam' or 'kayak'.

- Expected input:
In the first line, a single integer of size N, representing the amount 
of lines to read, followed by N lines containing a word.

- Expected output:
Print the analysis telling if the word is or isn't a palindrome

- Sample input:
3
madam
kayak
lick

- Sample output:
madam is palindrome
kayak is palindrome
lick is not palindrome

- Test cases:
All the test cases are in the 'tests' folder.
